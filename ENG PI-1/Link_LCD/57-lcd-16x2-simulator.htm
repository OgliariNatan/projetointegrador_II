DjLCDSIM este un JavaScript care simuleaza un afisaj LCD alfanumeric cu 16x2 caractere, cu controler HD44780 sau compatibil. Suporta toate instructiunile, mai putin read data/busy flag.

LCD-ul se poate controla si fara butoane, folosind script-uri JavaScript combinate cu urmatoarele functii:

  • instr(instructionCode) // sends instruction
  • data(caharacterCode) // sends data
  • mcl(x_times) // move cusror left
  • mcr(x_times) // move cursor right
  • sdl(x_times) // shift display left
  • sdr(x_times) // shift display right
  • DDRadrs(address) // set DDRAM address
  • CGRadrs(address) // set CGRAM address
  • putstr('string') // print string

Exemplu script LCD

//initialize LCD 16x2 and print a string on first LCD line
instr(0x38);
instr(0x0F);
instr(0x01);
instr(0x06);
putstr('www.bobtech.ro');
